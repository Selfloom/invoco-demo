Everything You Need to Bill Clients

Built for solo freelancers who want to track time, send invoices, and get paid — without the overhead.

Live Time Tracker

Start/stop timer with client assignment and notes. Built-in keyboard shortcut (Space). Tracks every billable minute with per-client hourly rates.

Client Management

Add clients with hourly rates, company info, and billing addresses. Assign time entries to clients. Data auto-populates invoices.

Invoice Generator

Select unbilled time entries, add line items, set tax rate, and generate professional invoices. Print directly or save as PDF from your browser.

Local Storage Only

All data stored in your browser's localStorage. No cloud, no servers, no tracking. Export/import your data as JSON for backup or migration.

Beautiful Dark Theme

Easy-on-the-eyes dark interface designed for long work sessions. Clean typography, smooth transitions, and responsive layout on any screen size.

Export & Import

One-click JSON export of all your data. Import on another device or keep versioned backups. Your data is never locked in.
